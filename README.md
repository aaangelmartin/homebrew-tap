# homebrew-tap

Homebrew tap for [@aaangelmartin](https://github.com/aaangelmartin)'s tools.

## Install

CLI tools (Formulae):

```sh
brew install aaangelmartin/tap/<formula>
```

GUI apps (Casks):

```sh
brew tap aaangelmartin/tap
brew install --cask <cask>
```

## Formulae

- [`aka`](https://github.com/aaangelmartin/aka) — manage shell aliases from the terminal with a TUI.
- [`goto`](https://github.com/aaangelmartin/GoTo) — open URLs and manage link aliases from the terminal.

## Casks

- [`podstats`](https://github.com/aaangelmartin/PodStats) — Stats, replays and playlist editor for the classic iPod (macOS).
